# 📸 Screenshot Guide

## Required Screenshots for README

To complete your portfolio README, capture the following screenshots:

### 1. Dashboard (dashboard.png)
**What to show:**
- Main dashboard view with all KPIs visible
- Revenue performance chart with data
- Top products list populated
- Critical stock alerts showing

**How to capture:**
1. Run the application
2. Go to Dashboard
3. Click "Generate Sample Data" button if database is empty
4. Click "Refresh" to ensure data is loaded
5. Take a full-window screenshot
6. Save as `dashboard.png` in this folder

**Recommended size:** 1920x1080 or 1280x720

---

### 2. Point of Sale (pos.png)
**What to show:**
- Sales view with products in cart
- Customer selected
- Total amount visible
- Clean, professional layout

**How to capture:**
1. Navigate to Sales view
2. Add 2-3 products to cart
3. Select a customer
4. Take screenshot before completing sale
5. Save as `pos.png`

---

### 3. Products View (products.png)
**What to show:**
- Product catalog in card layout
- Mix of active/inactive products
- Status badges visible
- At least 4-6 products showing

**How to capture:**
1. Go to Products view
2. Ensure you have several products with different statuses
3. Take screenshot showing the grid layout
4. Save as `products.png`

---

### 4. PDF Invoice (invoice.png)
**What to show:**
- A generated PDF invoice opened in PDF viewer
- Clear transaction details
- Professional formatting

**How to capture:**
1. Complete a sale
2. Generate PDF invoice
3. Open the PDF
4. Take screenshot of the PDF
5. Save as `invoice.png`

## Tips for Great Screenshots

✅ **Use a clean desktop background** (solid color or subtle gradient)
✅ **Maximize the window** for clarity
✅ **Ensure good contrast** - avoid dark screenshots
✅ **Show real data** not empty states
✅ **Crop unnecessary borders** but keep the window frame
✅ **Use consistent resolution** across all screenshots
✅ **Consider using a tool like Greenshot or Snagit** for better quality

---

## Screenshot Tools

**Windows Built-in:**
- `Win + Shift + S` - Snipping Tool (Windows 10/11)
- `Win + PrtScn` - Full screen to Pictures folder

**Recommended Tools:**
- [Greenshot](https://getgreenshot.org/) - Free, powerful
- [ShareX](https://getsharex.com/) - Free, feature-rich
- [Snagit](https://www.techsmith.com/screen-capture.html) - Professional (paid)

## After Capturing Screenshots

1. Save all images to this `docs/screenshots/` folder
2. Verify filenames match those in README.md:
   - `dashboard.png`
   - `pos.png`
   - `products.png`
   - `invoice.png`
3. Optionally compress images using [TinyPNG](https://tinypng.com/) to reduce file size
4. Commit and push to GitHub
5. Verify images display correctly on GitHub
